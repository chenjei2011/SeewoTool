[SeewoTool.md](https://github.com/user-attachments/files/23687325/SeewoTool.md)## 关于这个软件
> 正如你所看到的，这是一个seewo小工具，主要是解决一些作者班级上面seewo的问题和一些生活问题（校园），比如有些老师不会点击此电脑去选择自己的u盘，平时是靠360弹窗去开u盘的，然而，我把360卸载了，可能有老师不会打开u盘或习惯性等360弹窗出现，于是就有了u盘检测工具，又如听不到上课铃，所以有了上课提醒（目前没有做完），也真诚希望您能来这个项目中构建一些功能。
## 许可问题
> 承诺本软件能随意发行，更改。
## 构建环境
> 本软件用mingw编译运行，cmake构建项目，图形化依赖imgui，可通过git拉取，经测试完美适配windows7
